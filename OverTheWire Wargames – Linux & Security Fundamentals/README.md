### 🎯 Objective
Strengthen Linux command-line skills and web security fundamentals through hands-on Capture The Flag (CTF) challenges — building the foundational skills required for day-to-day SOC analyst work.

### 🛠️ Tools Used
- **Platform:** OverTheWire (overthewire.org)
- **Wargames Completed:** Bandit (Linux fundamentals) + Natas (Web security)
- **Environment:** Linux terminal (SSH)

### 🔧 What I Practised

**Bandit (Linux Fundamentals):**
- Linux command-line navigation (ls, cat, find, grep, sort)
- File permissions and ownership
- SSH key authentication
- Bash scripting basics
- Data encoding/decoding (base64, hex)
- Network connections (nc, telnet)
- Cron jobs and process management

**Natas (Web Security):**
- HTTP request/response analysis
- Cookie manipulation
- Source code review
- Basic SQL injection
- Directory traversal
- Authentication bypass

### 📊 Key Skills Demonstrated
`Linux CLI` `SSH` `File Permissions` `Bash` `Web Security` `HTTP Analysis` `SQL Injection basics` `CTF Problem Solving` `OWASP concepts`
